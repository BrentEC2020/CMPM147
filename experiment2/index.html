Experiment 2 - A Procedurally Generated Living Impression

Description

This program generates a living impression of a low-poly sunset using p5.js. The image makes use of gradients to imitate the color fades of the image and custom shapes to form mountains and rocks.

Technical

Relation to inspring image:

  • Color: The sky, ground, sun, mountain, rocks, and star colors were sampled from the inspiring image using an online eyedropper tool.
  • Primitives: The stones are rendered with quad primitives.
  • Polygons: The mountains are created using three different custom polygons using vertex calls.
  • Composition: Different random seed values generate different mountain polygons, varied positioned rocks, and the number and movement of the stars.
  • Life: The rocks respond to the movement of the user's mouse cursor to replicate the experience of walking by the scene.
